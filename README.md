# 欢迎使用文档中心

> 这是一个基于 docsify 的文档中心

## 快速开始

1. 克隆项目
2. 安装依赖
3. 运行项目

## 项目结构

```
.
├── README.md
├── _navbar.md
├── _sidebar.md
├── index.html
└── docs/
    └── guide/
        └── README.md
```

## 功能特性

- 支持 Markdown
- 支持代码高亮
- 支持全文搜索
- 支持图片缩放
- 支持代码复制
